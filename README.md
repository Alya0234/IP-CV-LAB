## 📌 About This Repository
This repository contains implementations of the first 10 experiments from the Image Processing & Computer Vision Lab. Each experiment demonstrates fundamental concepts ranging from image acquisition to edge detection and compression.

---

## 🧠 Tools & Technologies
- Python
- OpenCV
- NumPy
- Matplotlib

---

## 📂 Experiments Overview

---

### 🔹 Experiment 1: Image Acquisition
- Loaded and displayed an image using OpenCV.
- Understood image sensing process (CCD, CMOS, ADC).
- Learned how real-world scenes are converted into digital images.

---

### 🔹 Experiment 2: Sampling & Quantization
- Reduced image resolution using sampling.
- Reduced intensity levels using quantization.
- Observed effects like blurring and intensity banding.

---

### 🔹 Experiment 3: Image Enhancement
- Applied contrast stretching.
- Used histogram equalization.
- Improved image visibility and contrast.

---

### 🔹 Experiment 4: Frequency Domain Filtering
- Converted image to frequency domain using FFT.
- Applied:
  - Low-pass filter → smoothing
  - High-pass filter → edge enhancement

---

### 🔹 Experiment 5: Noise & Restoration
- Added:
  - Gaussian noise
  - Salt & pepper noise
- Applied:
  - Mean filter
  - Median filter
- Compared restoration effectiveness.

---

### 🔹 Experiment 6: Periodic Noise Removal
- Simulated periodic noise.
- Applied Wiener filtering.
- Improved image quality by minimizing noise.

---

### 🔹 Experiment 7: Geometric Transformations
- Performed:
  - Translation
  - Scaling
  - Rotation
- Observed spatial transformation effects.

---

### 🔹 Experiment 8: Color Image Processing
- Converted RGB → HSV color space.
- Applied color-based segmentation.
- Extracted regions based on color ranges.

---

### 🔹 Experiment 9: Image Compression
- Implemented Run-Length Encoding (RLE).
- Reduced storage by encoding repeated pixels.
- Calculated compression efficiency.

---

### 🔹 Experiment 10: Edge Detection
- Applied Canny edge detection.
- Identified object boundaries.
- Used edges for segmentation.

---

## 📊 Key Learnings
- Digital images are matrices of pixel values.
- Image quality depends on sampling and quantization.
- Noise can be reduced using appropriate filters.
- Segmentation and edge detection help identify objects.
- Compression reduces storage without losing essential information.
